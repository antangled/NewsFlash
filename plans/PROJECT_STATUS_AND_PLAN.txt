OVERVIEW
--------
NewsFlash is a Chrome extension + Node.js backend that delivers high-signal
tech news from Twitter/X passively via a frosted-glass bottom bar on every
webpage. Free tier gives 5 stories/day; Pro ($4.99/mo) unlocks 10 stories,
breaking alerts, archive access, and topic customization.

Architecture: Express 5 + TypeScript + SQLite + Gemini 2.0 Flash (backend)
              Manifest V3 + Shadow DOM content script + popup (extension)
              RSSHub Docker container for Twitter/X feed ingestion

                        QUICK START (for dev)
==============================================================================

  1. Start Docker Desktop
  2. Run: ./start.sh
     (starts RSSHub container + backend dev server)
  3. Load extension in Chrome:
     chrome://extensions → Load unpacked → select extension/ folder
  4. To test pipeline manually:
     curl -X POST http://localhost:3000/api/pipeline/run \
       -H "X-NewsFlash-Key: nf-dev-key-2024"

==============================================================================
                      DATA FLOW (end to end)
==============================================================================

  Twitter/X accounts (57 curated)
       ↓
  RSSHub (Docker, port 1200) converts to RSS
       ↓
  [Ingest] fetches last 24h tweets → raw_tweets table
       ↓
  [Filter] removes spam/noise (~50-60% filtered)
       ↓
  [Score] Gemini classifies events + tier weighting
       ↓
  [Cluster] Gemini groups by topic + cross-source bonus
       ↓
  [Summarize] Gemini generates headlines + details → stories table
       ↓
  Extension fetches /api/stories/today (every 60 min)
       ↓
  Content script injects bottom bar (shadow DOM, every page)
       ↓
  User reads headlines passively, clicks to expand
